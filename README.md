# Member4s
一个VUE 类似手风琴特效的路由切换
如果对您有帮助，您可以点右上角 "Star" 支持一下 谢谢！ ^_^

环境

node v6.2.0
cnpm 4.3.2
npm 3.8.9

通过npm安装本地服务第三方依赖模块(需要已安装Node.js)，使用npm安装依赖模块可能会很慢，建议换成cnpm
npm install -g cnpm --registry=http://registry.npm.taobao.org

# 安装依赖模块
cnpm install

# 启动服务
npm run dev

# 发布代码
npm run build
